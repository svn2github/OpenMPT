Sample Window

The following commands are available in the sample window:
  • New Sample
  • Import Sample
  • Export Sample
  • Test Sample
  • Normalize
  • Amplify
  • Upsample
  • Downsample
  • Reverse

New Sample

This command, available in the sample toolbar, inserts a new sample in the song (It doesn't overwrite the current sample).
There is a limit of 199 samples in a song.

Import Sample

This command allows you to import a sample from a WAV,PAT,XI,SND,AIFF file. (You can also load any file as raw sample data).

Export Sample

This command allows you to export the current sample as a WAVE file.

Test Sample

This small button plays a middle C (C-5) using the current sample, or stops it if the sample is currently playing.

Normalize

This function will normalize the current sample, so it will use the full-scale 8-bit or 16-bit, and having the best possible SNR. Always try to use normalized samples: if it sounds too loud, adjust the global volume of the sample instead.

Amplify

This function will amplify the current sample. It is better to used the Normalize function, since amplifying too much a sample can make it clip. You can also use this function for fade-in and fade-out effects.

Upsample

The upsample function will stretch the current selection, and convert the sample to 16-bit

Upsample

The downsample function will shrink the current selection

Reverse

This command will reverse the current sample
